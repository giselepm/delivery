## BUILDING PROJECT
This solution uses [Gradle](https://gradle.org/) as build mechanism.

Type:

    ./gradlew clean test

Downloaded files (including the Gradle distribution itself) will be stored in the Gradle user home directory (typically *user_home*`/.gradle`).
